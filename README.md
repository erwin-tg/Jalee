# Jalee
Sketch master et slave pour le contrôle du robot-chien Jalee à l'aide d'une Arduino Uno et Arduino Pro Mini (protocole I2C)
Arduino Uno (master) Elegoo Uno
  - Contrôle moteur TT 1:120 avec encodeur intégré FIT0450 (x2) 
  - Contrôle joystick AM-JOYSTICK Module analogique de manette à balai
  - Contrôle écran LCD, GK-OLED-0.96 geeekus 0.96'' Module d'affichage bleu OLED128X64 Série I2C pour Arduino
Arduino Pro Mini (slave) 5V 16 MHz
  - Contrôle sub-microservo
  - Contrôle servomoteur MG996R
